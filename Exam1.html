Exam 1

Create a JSTest class (using prototype) which has the following methods:
  • getPrimeNumbers: Which prints the first N ‘prime’ numbers (N is a value provided by the user)
  • getFactorial: Which calculates the factorial of a given number (number provided by the user).
  • countVowels: Which counts the number of vowels ‘a’, ‘e’, ‘i’, ‘o’ and ‘u’ that a given paragraph has (paragraph provided by the user).
  • evalParagraph: Which, given a long paragraph and a letter/char provided by the user, print the count and all the words that contain the specified char.
  • Example. var paragraph = ‘hello world’;
    var letter = ‘h’;
    ⇒ Count is: 1
    ⇒ Words: hello
  • Create a separate file which has an auto-executable function in order to run all methods (all 4 above exercises) from JSTest class
